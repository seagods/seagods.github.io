Video Tutorials

For non MS windows people, and MS windows people inexperienced in programming, I have an introductory video series Building Programs and Libraries on MS Windows Starts with very basic windows, covers installing and using compilers (especially GCC --- the Gnu Compiler Collection) and installing useful things with MSYS2. You will, for instance, be able to download and build fortran libraries and mix programming languages.. Note the emphasis is on "building programs" rather than writing them. (No programing language tutorials here.)
