"Free Compilers on MS Windows"

If you are on a linux operating system, you have the world class GNU Compiler Collection (GCC) freely available. You can use GCC with MS windows using MinGW or Cygwin . The problem is that microsoft isn't POSIX compliant, whereas GCC complies to the international POSIX standard.

If you don't want to go down the MinGW/Cygwin route, there are free compilers for MS windows. These are for personal and non-commercial use only.

At the time of writing this, the MS windows visual express suite contains visual C++, visual C#, visual basic, and web developer software. The express editions of 2010 and 2008 are free. You want the 2008 edition if you want to include an express edition of the Salford FTN95 FORTRAN compiler. The FORTRAN compiler in the list below includes C++. This means that you can call many excellent (and free) mathematical FORTRAN routines from your C/C++ programs. Such mathematical FORTRAN routines are available from Netlib .

These are the links for the free 2008 visual studio and FORTRAN compilers that run on MS windows

  • Visual Studio 2008 express edition
  • FTN95 Compiler (includes a C++ compiler)

I shall leave it to you to find out the exactly what personal use means and non commercial means. I shall mention the conditions are quite strict. Nonetheless I can say, that if you install these on your home machine, and you are not using the compilers as part of your job you are OK. However, what happens if you decide that something you have written has commercial value? I don't know, it's up to you to find out if things turn out that way later.
