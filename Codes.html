Chris Godsalve's Computer Codes

I have moved all my downloadable codes to GitHub. So, the first thing you need to do is get GIT (General Information Tracker). This is Linus Torvald's tool for following the development of code by multiple programmers at different locations (a version control system). You can get GIT HERE . To get my program (or set of programs) X from GitHub you need to open up a command line tool and type
"git clone git*github.com:seagods/X.git"

Download Some of my Computer Programmes

  • X=Geom, for looking at the Dodecahedron, Icosahedron, and Truncated Icosahedron
  • X=Games, a simple terrain explorer and Flight Simulator is all that's here so far
  • X=PlotFiles, a set of 2D and 3D graphics programs with documentation.
  • X=bitmaps, bitmaps needed by all the PlotFiles and Games.
  • X=XFiles, headers needed by all the PlotFiles and Games.
  • X=MieOhMy, my own Mie scattering programs (needs MyLib - see below).
  • X=SphereTree, a set of programs for quadtrees on spheres (see Computational Maths ).
  • X=MyLib, hacks of old fortran programs that will now compile with gfortran, needed by MieOhMy (see See Guide to Available Mathematical Software (GAMS) and NETLIB
  • X=Geodesy, Work in progress, geodisics on triangulated curved surfaces.
